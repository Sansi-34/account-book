# Contributing

欢迎改进账号本。这个项目优先保持简单、透明、可本地使用。

## 开发原则

- 默认不接入服务器
- 不加入统计、广告或第三方追踪
- 安全相关改动要写清楚理由
- 体验改动尽量保持朴素、实用
- 不承诺“绝对安全”

## 提交问题

提交 issue 时，请尽量说明：

- 使用的浏览器和版本
- 操作系统
- 复现步骤
- 期望结果
- 实际结果

如果问题和安全有关，请先看 `SECURITY.md`。

## 本地检查

项目没有构建步骤。提交前建议运行：

```bash
npm test
```

测试会检查基本加密流程：

- 正确密码可以解密
- 错误密码不能解密
- 解密后的记录和原记录一致
